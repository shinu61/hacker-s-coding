# hacker-s-coding
White Hat School 해커의 프로그래밍 과제

## 과제 목표
`http://board.nyan101.com/` 게시판의 모든 댓글을 수집해
사용자별 활동 시간대를 분석하고 비정상적인 계정을 탐지한다.

## 파일 구성
| 파일 | 설명 |
|---|---|
| `q2.py` | 비정상 활동패턴 사용자 자동 탐지 및 시각화 |
| `q2_output/` | 비정상 유저 히트맵 PNG 결과물 |

## Q2 - 비정상 활동패턴 탐지

### 탐지 기준

**기준 1 - 24시간 활성화**
최대 연속 휴식 시간이 180분 미만이면 비정상으로 판단.
인간은 생체 리듬상 최소 3시간 이상의 수면/휴식이 반드시 발생하기 때문.

**기준 2 - 1분 도배**
동일한 1분 안에 댓글 5개 이상이면 비정상으로 판단.
사람이 1분 안에 5개 이상의 댓글을 작성하는 것은 물리적으로 불가능.

**기준 3 - 고정 간격**
댓글 작성 간격(분)의 표준편차가 1.5 미만이면 비정상으로 판단.
봇은 타이머로 동작하기 때문에 간격이 거의 일정하게 유지됨.

**기준 4 - 댓글 내용 중복**
동일한 댓글 내용이 3회 이상 반복되면 비정상으로 판단.

**기준 5 - 정각/5분 배수 집중**
전체 댓글 중 작성 분(minute)이 5의 배수인 비율이 80% 이상이면 비정상으로 판단.
